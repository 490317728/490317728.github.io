    reader项目-day5

    4、列表页与详情页跳转

    （1）使用 “…item” 来展开数据元素，在调用时，不需要再前面加 “item.”

    <template is="postItem" data="{{...item}}" />

    <template name="postItem">
  <view class='post-container'>
    <view class='post-author-date'>
      <image class='post-author' src='{{avatar}}'></image>
      <text class='post-date'>{{date}}</text>
    </view>
  </view>
</template>

    (2) 在文章列表页中点击不同文章可以跳转到对应的文章的详情页，通过id的方式

    首先，数据库中每个文章需要有一个id，例：postId

    其次，在wxml文件中获取postId，data-xxx为自定义属性，再点击时会进行调用

    <view bindtap='onPostTap' data-postId='{{item.postId}}'>
      <template is="postItem" data="{{...item}}" />
  </view>

    最后，在js文件中，通过方法调用的方式，获得点击所对应的id

    var postId = event.currentTarget.dataset.postid

      onPostTap:function(event){
    var postId = event.currentTarget.dataset.postid;
    wx.navigateTo({
      url: 'post-detail/post-detail',
    })
  }

    （3）页面跳转，相关操作

    需要在app.json文件中添加相应页面。

      "pages": [
    "pages/index/index",
    "pages/posts/posts",
    "pages/posts/post-detail/post-detail"
  ],

    此时，跳转页面的框架已经搭好。
