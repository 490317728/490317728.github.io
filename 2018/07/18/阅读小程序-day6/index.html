    5、详情页数据填充

    （1）通过postId来指定每一个详情页的内容

    在列表页的js中获取了每篇文章的postId，

    再点击列表页时，把postId传递到详情页中

        /posts.js
    wx.navigateTo({
      url: 'post-detail/post-detail?id='+postId,
    })

    (2)在详情页的js文件中，接受到来自列表页的id请求并在数据库中寻找数据

    /post-detail.js
  onLoad: function (option) {
    var postId = option.id;
    var postData = postsData.postList[postId];
    this.setData({
      postDetail:postData
    })
  },

    option.id 会自动获取传入进来的id

    postsData.postList[postId] 获取数据库中指定id的所有数据内容

    this.setData({})绑定数据，并传入wxml中
