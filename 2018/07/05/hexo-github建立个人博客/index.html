    安装：

    1、 安装node.js（到官网下载最新的稳定版本，老版本亲测有问题）

    2、 安装git（官网下载）

    3、 安装hexo

    在gitbash中输入：

    $ npm install -g hexo

    4、 新建一个文件夹并进入：比如 /hexo

    $ cd hexo

    5、 初始化

    $ hexo init

    6、 生成生成静态页面至public目录

    $ hexo g

    7、 开启服务

    $ hexo s

    8、 浏览器输入localhost:4000

    连接：

    配置_config.yml中有关deploy的部分

    deploy:
    type: git
    repository: git@github.com:490317728/490317728.github.io.git
    branch: master

    安装插件

    $ npm install hexo-deployer-git –save

    部署到github

    $ hexo d

    博客内容

    创建博客

    $ hexo new “postName” #新建文章

    删除博客

    删除本地文件

    $ hexo g
    $ hexo d
