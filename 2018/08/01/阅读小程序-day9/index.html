    reader项目-day8

    5、详情页–音乐播放功能实现

    （1）在wxml中添加方法 catchtap=’onMusicTap’，并在js文件中编写方法

    在onload中的data里添加isPlayingMusic变量

onMusicTap: function (event) {
    var isPlayingMusic = this.data.isPlayingMusic
    var currentPostId = this.data.currentPostId
    var postData = postsData.postList[currentPostId]
    if (isPlayingMusic) {
      wx.pauseBackgroundAudio()
      this.setData({
        isPlayingMusic: false,
        postData:postData
      })
    } else {
      wx.playBackgroundAudio({
        dataUrl: postData.music.url,
        title: postData.music.title,
        coverImgUrl: postData.music.coverImg,
      })
      this.setData({
        isPlayingMusic: true,
        postData: postData
      })
    }
  }

    wx.playBackgroundAudio（）使用后台播放器播放音乐，对于微信客户端来说，只能同时有一个后台音乐在播放。当用户离开小程序后，音乐将暂停播放；当用户在其他小程序占用了音乐播放器，原有小程序内的音乐将停止播放。

    dataUrl 音乐链接，目前支持的格式有 m4a, aac, mp3, wav

    title 音乐标题

    coverImgUrl 封面URL

    wx.pauseBackgroundAudio()暂停播放音乐。

    （2）使用音乐监听api，实现后台音乐播放与页面音乐播放同步。

        wx.onBackgroundAudioPlay(function(){
      that.setData({
        isPlayingMusic:true
      })
    })
    wx.onBackgroundAudioPause(function(){
      that.setData({
        isPlayingMusic: false
      })
    })

    wx.onBackgroundAudioPlay(CALLBACK)监听音乐播放。

    wx.onBackgroundAudioPause(CALLBACK)监听音乐暂停。

    当后台音乐播放或者暂停时，设置页面也为播放或者暂停

    （3）当退出页面时，音乐仍然播放，但是在进入此页面，音乐继续播放，图片又恢复最原始状态，不再是播放状态。

    此时需要加入全局变量去实现监听功能

    首先在app.js中添加全局变量
 globalData: {
    g_isPlayingMusic:false,
  }

    然后在post-detail.js顶部引入全局变量
var app = getApp()

     var that = this
if (app.globalData.g_isPlayingMusic){
      that.setData({
        isPlayingMusic: true
      })
    } 
wx.onBackgroundAudioPlay(function(){
      that.setData({
        isPlayingMusic:true
      })
      app.globalData.g_isPlayingMusic = true;
})
wx.onBackgroundAudioPause(function(){
      that.setData({
        isPlayingMusic: false
      })
      app.globalData.g_isPlayingMusic = false;
})

    如果监听到音乐正在播放，则设置全局变量为true，当下次进入时，首先判断全局变量，如果为true，则设置页面为正在播放状态

    （4）当播放一首音乐并跳转到另一个页面时，另一个页面仍然显示正在播放的音乐的样式。

    此时需要新的全局变量来判断所在页面

    在app.js中添加全局变量
 globalData: {
    g_isPlayingMusic:false,
    g_currentMusicPostId:null,
  }

     var that = this
    if (app.globalData.g_isPlayingMusic && app.globalData.g_currentMusicPostId === postId){
      that.setData({
        isPlayingMusic: true
      })
    }

    wx.onBackgroundAudioPlay(function(){
      that.setData({
        isPlayingMusic:true
      })
      app.globalData.g_isPlayingMusic = true;
      app.globalData.g_currentMusicPostId = this.data.currentPostId;
    })
    wx.onBackgroundAudioPause(function(){
      that.setData({
        isPlayingMusic: false
      })
      app.globalData.g_isPlayingMusic = false;
      app.globalData.g_currentMusicPostId = null;
    })

    如果某页面中的音乐正在播放时，则设置全局变量为当前页面的id

    这时如果还是当前页面，则页面播放样式不变

    如果到其他页面，则会设置全局变量的id为空
