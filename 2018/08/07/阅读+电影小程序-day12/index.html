    7.Movie页–api调用（调用豆瓣的api文档）

    (1)在movie.js文件中就行调用

    onLoad:function(event){
    var inTheaterUrl = app.globalData.douban_base + "/v2/movie/in_theaters"+"?start=0&count=3";
    var inComingUrl = app.globalData.douban_base + "/v2/movie/coming_soon" + "?start=0&count=3";
    var top250Url = app.globalData.douban_base + "/v2/movie/top250" + "?start=0&count=3";

    this.getMovieListData(inTheaterUrl);
    this.getMovieListData(inComingUrl);
    this.getMovieListData(top250Url);
  },

    inTheaterUrl，inComingUrl，top250Url为调用的url路径

    app.globalData.douban_base为在app.js中定义的全局变量

     globalData: {
    douban_base: 'http://t.yushu.im',
  }

    ‘/v2/movie/in_theaters’ 为豆瓣api路径

    “?start=0&count=3” 表示从0开始取数据，取三个数据

    注意：需要在movie.js中引入全局变量

    var app = getApp()

    (2)定义一个getMovieListData方法，这个方法就是具体调用指定url数据的方法

    getMovieListData:function(url){
    wx.request({
      url: url,
      method: "GET",
      header: {
        "Content-Type": "json"
      },
      success: function (res) {
        console.log(res)
      },
      fail: function (error) {
        console.log(error)
      }
    })
  }

    method: “GET”,是指获取数据的方法，还有”PUT”,”PUSH”,”DELETE”,”OPTION”等方法

    header为数据请求必须的项

    success方法为数据获取成功时的输出

    fail则是指网络连接失败后提示的失败
