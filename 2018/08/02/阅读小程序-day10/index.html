    reader项目-day10（1）

    5、详情页–轮播图跳转

    首先在wxml中设定对应轮播图的id，并绑定方法

    <swiper catchtap='onSwiperTap' autoplay='true' interval='3000' indicator-dots='true'>
    <swiper-item>
      <image data-postId="5" src='/images/iqiyi.png'></image>
    </swiper-item>
    <swiper-item>
      <image data-postId="4" src='/images/vr.png'></image>
    </swiper-item>
    <swiper-item>
      <image data-postId="3" src='/images/wx.png'></image>
    </swiper-item>
  </swiper>

    接着在js文件中设定方法

      onSwiperTap:function(event){
    var postId = event.target.dataset.postid;
    wx.navigateTo({
      url: 'post-detail/post-detail?id=' + postId,
    })
  }

    target 和 currentTarget的区别

    target指的是当前点击的组件 currentTarget指的是事件捕获的组件

    target这里指的是image，而currentTarget指的是swiper
