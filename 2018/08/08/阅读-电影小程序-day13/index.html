    Reader+Movie项目-day13

    7.Movie页–数据绑定

    （1）getMovieListData获取数据，获取成功后，在success方法中绑定数据

     getMovieListData:function(url){
    var that = this
    wx.request({
      url: url,
      method: "GET",
      header: {
        "Content-Type": "json"
      },
      success: function (res) {
        console.log(res)
        that.processDoubanData(res.data)
      },
      fail: function (error) {
        console.log(error)
      }
    })
  },

    定义一个processDoubanData的方法用来处理数据（传递参数res.data）

    (2)在processDoubanData方法中处理数据

    processDoubanData: function (moviesDouban, settedKey){
    var movies = [];
    for (var idx in moviesDouban.subjects){
      var subject = moviesDouban.subjects[idx];
      var title = subject.title;
      if(title.length >= 6){
        title = title.substring(0,6) + "...";
      }
      var temp = {
        title:title,
        average:subject.rating.average,
        coverageUrl:subject.images.large,
        movieId:subject.id,
      }
      movies.push(temp)
    }
    this.setData({
     movies:movies   
    })
}

    for循环获取每一个电影id，并通过这个电影id把这个电影的所有数据赋值给subject变量

    if判断电影名字长度，如果大于6，则通过substring方法截取

    通过一个temp变量存储这一大类中，所有需要的数据，并放到movies变量中

    通过setData绑定前端数据

    以上只是绑定一个大类中的电影数据内容，下面则是详细的绑定三个大类中的内容

    （3）在data中定义三个大类的数据集

      data:{
    inTheaters:{},
    ComingSoon:{},
    top250:{}
  },

    （4）在onload函数，调用getMovieList方法中，指定传入的参数

      onLoad:function(event){
    var inTheaterUrl = app.globalData.douban_base + "/v2/movie/in_theaters"+"?start=0&count=3";
    var inComingUrl = app.globalData.douban_base + "/v2/movie/coming_soon" + "?start=0&count=3";
    var top250Url = app.globalData.douban_base + "/v2/movie/top250" + "?start=0&count=3";

    this.getMovieListData(inTheaterUrl,"inTheaters");
    this.getMovieListData(inComingUrl,"ComingSoon");
    this.getMovieListData(top250Url,"top250");
  },

    （5）在getMovieListData方法中接受参数，并把参数传递到processDoubanData方法中

        getMovieListData:function(url,settedKey){
        .....
    success: function (res) {
        console.log(res)
        that.processDoubanData(res.data, settedKey)
      },
    }

    processDoubanData: function (moviesDouban, settedKey){
    var movies = [];
    for (var idx in moviesDouban.subjects){
      var subject = moviesDouban.subjects[idx];
      var title = subject.title;
      if(title.length >= 6){
        title = title.substring(0,6) + "...";
      }
      var temp = {
        title:title,
        average:subject.rating.average,
        coverageUrl:subject.images.large,
        movieId:subject.id,
      }
      movies.push(temp)
    }
    var readyData = {};
    readyData[settedKey] ={
      movies:movies
    } 
    this.setData(readyData)
  }

    readyData[settedKey] 动态绑定数据

    this.setData（readyData）设置数据到data：中指定的数据集中

    （6）在movies.wxml中接受绑定的数据

    data=”…inTheaters”为传递inTheaters数据到movieListTemplate界面。…为铺开数据

    （7）在movie-list-template.wxml文件中修改

    <block wx:for="{{movies}}" wx:for-item="movie">
    <template is="movieTemplate" data="{{...movie}}" />
</block>

    对每个电影进行数据绑定，data=”…movie”所以传递每个电影到movieTemplate
